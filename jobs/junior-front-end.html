Junior Web Front-End Developer

New Branch Ltd is looking for its first employee!

The Ideal candidate should be passionate about software development and especially about implementing beautiful, functional and high quality user interfaces. The applicant has to be eager for learning new technologies, tools, practices and be willing to put extra effort for this.

  • Ability to work unsupervised and without full specification
  • Focused on the added value for the internal and/or external clients
  • Flexible, adaptable, open to meet the change and acquire new knowledge and skills

Bonus Requirements

  • Familiarity with version control systems and specifically with GIT
  • You are a happy Linux user

  • Motivating compensation + bonuses for performance

  • Be part of an attempt to build a great software development company
